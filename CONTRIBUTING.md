# Contributing to FeedbackKit Vapor SDK

Thank you for your interest in contributing to FeedbackKit! We welcome contributions from the community.

## How to Contribute

### Reporting Bugs

If you find a bug, please [open an issue](https://github.com/Swiftly-Developed/SwiftlyFeedbackKit-Vapor-SDK/issues) with:

- A clear description of the issue
- Steps to reproduce
- Expected vs actual behavior
- SDK version and Vapor/Swift version details

### Suggesting Features

Feature requests are welcome! Please [open an issue](https://github.com/Swiftly-Developed/SwiftlyFeedbackKit-Vapor-SDK/issues) with:

- A clear description of the feature
- The use case it addresses
- Any relevant examples or mockups

### Pull Requests

1. **Fork** the repository
2. **Create a feature branch** from `dev` (e.g., `feature/my-new-feature`)
3. **Follow existing code conventions** in the project
4. **Write clear commit messages** following conventional commits (e.g., `feat:`, `fix:`, `chore:`)
5. **Submit a pull request** to the `dev` branch with a description of your changes

### Development Setup

```bash
swift build
swift test
```

See the [README](README.md) for detailed setup instructions.
